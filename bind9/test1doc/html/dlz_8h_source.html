139 /*%<
140  * Method prototype. Drivers implementing the DLZ interface MUST
141  * supply a find zone method. This method is called when the DNS
142  * server is performing a query. The find zone method will be called
143  * with the longest possible name first, and continue to be called
144  * with successively shorter domain names, until any of the following
145  * occur:
146  *
147  * \li 1) a match is found, and the function returns (ISC_R_SUCCESS)
148  *
149  * \li 2) a problem occurs, and the functions returns anything other
150  * than (ISC_R_NOTFOUND)
151  * \li 3) we run out of domain name labels. I.E. we have tried the
152  * shortest domain name
153  * \li 4) the number of labels in the domain name is less than
154  * min_labels for dns_dlzfindzone
155  *
156  * The driver's find zone method should return ISC_R_SUCCESS and a
157  * database pointer to the name server if the zone is supported by the
158  * database. Otherwise it will return ISC_R_NOTFOUND, and a null
159  * pointer if the zone is not supported. If an error occurs it should
160  * return a result code indicating the type of error.
161  */

276 /*%<
277  * This method is called when the name server is starting up to parse
278  * the DLZ driver command line from named.conf. Basically it splits
279  * up a string into and argc / argv. The primary difference of this
280  * method is items between braces { } are considered only 1 word. for
281  * example the command line "this is { one grouped phrase } and this
282  * isn't" would be parsed into:
283  *
284  * \li argv[0]: "this"
285  * \li argv[1]: "is"
286  * \li argv{2]: " one grouped phrase "
287  * \li argv[3]: "and"
288  * \li argv[4]: "this"
289  * \li argv{5}: "isn't"
290  *
291  * braces should NOT be nested, more than one grouping in the command
292  * line is allowed. Notice, argv[2] has an extra space at the
293  * beginning and end. Extra spaces are not stripped between a
294  * grouping. You can do so in your driver if needed, or be sure not
295  * to put extra spaces before / after the braces.
296  */
