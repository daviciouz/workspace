632 /*%<
633  * Open a new socket file descriptor of the given socket structure. It simply
634  * opens a new descriptor; all of the other parameters including the socket
635  * type are inherited from the existing socket. This function is provided to
636  * avoid overhead of destroying and creating sockets when many short-lived
637  * sockets are frequently opened and closed. When the efficiency is not an
638  * issue, it should be safer to detach the unused socket and re-create a new
639  * one. This optimization may not be available for some systems, in which
640  * case this function will return ISC_R_NOTIMPLEMENTED and must not be used.
641  *
642  * isc_socket_open() should not be called on sockets created by
643  * isc_socket_fdwatchcreate().
644  *
645  * Requires:
