Detailed Description

A DNS rdataset is a handle that can be associated with a collection of rdata all having a common owner name, class, and type.

The dns_rdataset_t type is like a "virtual class". To actually use rdatasets, an implementation of the method suite (e.g. "slabbed rdata") is required.
