291 /*%<
292  * Recurse to answer a question.
293  *
294  * Notes:
295  *
296  *\li This call starts a query for 'name', type 'type'.
297  *
298  *\li The 'domain' is a parent domain of 'name' for which
299  * a set of name servers 'nameservers' is known. If no
300  * such name server information is available, set
301  * 'domain' and 'nameservers' to NULL.
302  *
303  *\li 'forwarders' is unimplemented, and subject to change when
304  * we figure out how selective forwarding will work.
305  *
306  *\li When the fetch completes (successfully or otherwise), a
307  * #DNS_EVENT_FETCHDONE event with action 'action' and arg 'arg' will be
308  * posted to 'task'.
309  *
310  *\li The values of 'rdataset' and 'sigrdataset' will be returned in
311  * the FETCHDONE event.
312  *
313  *\li 'client' and 'id' are used for duplicate query detection. '*client'
314  * must remain stable until after 'action' has been called or
315  * dns_resolver_cancelfetch() is called.
316  *
317  * Requires:
318  *
319  *\li 'res' is a valid resolver that has been frozen.
320  *
321  *\li 'name' is a valid name.
322  *
323  *\li 'type' is not a meta type other than ANY.
324  *
325  *\li 'domain' is a valid name or NULL.
326  *
327  *\li 'nameservers' is a valid NS rdataset (whose owner name is 'domain')
328  * iff. 'domain' is not NULL.
329  *
330  *\li 'forwarders' is NULL.
331  *
332  *\li 'client' is a valid sockaddr or NULL.
333  *
334  *\li 'options' contains valid options.
335  *
336  *\li 'rdataset' is a valid, disassociated rdataset.
337  *
338  *\li 'sigrdataset' is NULL, or is a valid, disassociated rdataset.
339  *
340  *\li fetchp != NULL && *fetchp == NULL.
341  *
342  * Returns:
343  *
344  *\li #ISC_R_SUCCESS Success
345  *\li #DNS_R_DUPLICATE
346  *\li #DNS_R_DROP
347  *
348  *\li Many other values are possible, all of which indicate failure.
349  */
