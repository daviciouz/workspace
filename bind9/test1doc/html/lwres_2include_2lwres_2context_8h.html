Allocate a lwres context. This is used in all lwres calls.

Memory management can be replaced here by passing in two functions. If one is non-NULL, they must both be non-NULL. "arg" is passed to these functions.

Contexts are not thread safe. Document at the top of the file. XXXMLG

If they are NULL, the standard malloc() and free() will be used.

Precondition
contextp != NULL && contextp == NULL.
Returns
Returns 0 on success, non-zero on failure.
