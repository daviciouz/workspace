Create and send a request.

Notes:

  • 'msgbuf' will be sent to 'destaddr' after setting the id. If the #DNS_REQUESTOPT_TCP option is set, TCP will be used, #DNS_REQUESTOPT_SHARE option is set too, connecting TCP (vs. connected) will be shared too. The request will timeout after 'timeout' seconds. UDP requests will be resent at 'udptimeout' intervals if non-zero or if 'udpretries' is not zero.
  • When the request completes, successfully, due to a timeout, or because it was canceled, a completion event will be sent to 'task'.

Requires:

  • 'msgbuf' is a valid DNS message in compressed wire format.
  • 'destaddr' is a valid sockaddr.
  • 'srcaddr' is a valid sockaddr or NULL.
  • 'srcaddr' and 'dstaddr' are the same protocol family.
  • 'timeout' > 0
  • 'task' is a valid task.
  • requestp != NULL && *requestp == NULL
