Detailed Description

Defines dns_badcache_t, the "bad cache" object.

Notes:

  • A bad cache object is a hash table of name/type tuples, indicating whether a given tuple known to be "bad" in some sense (e.g., queries for that name and type have been returning SERVFAIL). This is used for both the "bad server cache" in the resolver and for the "servfail cache" in the view.
