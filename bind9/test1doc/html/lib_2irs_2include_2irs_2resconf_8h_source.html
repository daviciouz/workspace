39 /*%<
40  * Load the resolver configuration file 'filename' in the "resolv.conf" format,
41  * and create a new irs_resconf_t object from the configuration. If the file
42  * is not found ISC_R_FILENOTFOUND is returned with the structure initialized
43  * as if file contained only:
44  *
45  * nameserver ::1
46  * nameserver 127.0.0.1
47  *
48  * Notes:
49  *
50  *\li Currently, only the following options are supported:
51  * nameserver, domain, search, sortlist, ndots, and options.

53  * the application cannot use the configuration.
54  *
55  * Returns:
56  * \li ISC_R_SUCCESS on success

60  * Requires:
