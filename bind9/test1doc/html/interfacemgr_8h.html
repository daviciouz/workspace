Detailed Description

The interface manager monitors the operating system's list of network interfaces, creating and destroying listeners as needed.

Reliability:

  • No impact expected.

Resources:

Security:

  • The server will only be able to bind to the DNS port on newly discovered interfaces if it is running as root.

Standards:

  • The API for scanning varies greatly among operating systems. This module attempts to hide the differences.
