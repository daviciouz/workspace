71 /*%<
72  * Create a new DNS cache.
73  *
74  * dns_cache_create2() will create a named cache.
75  *
76  * dns_cache_create3() will create a named cache using two separate memory
77  * contexts, one for cache data which can be cleaned and a separate one for
78  * memory allocated for the heap (which can grow without an upper limit and
79  * has no mechanism for shrinking).
80  *
81  * dns_cache_create() is a backward compatible version that internally
82  * specifies an empty cache name and a single memory context.
83  *
84  * Requires:
85  *
86  *\li 'cmctx' (and 'hmctx' if applicable) is a valid memory context.
87  *
88  *\li 'taskmgr' is a valid task manager and 'timermgr' is a valid timer
89  * manager, or both are NULL. If NULL, no periodic cleaning of the
90  * cache will take place.
91  *
92  *\li 'cachename' is a valid string. This must not be NULL.
93  *
94  *\li 'cachep' is a valid pointer, and *cachep == NULL
95  *
96  * Ensures:
97  *
98  *\li '*cachep' is attached to the newly created cache
99  *
100  * Returns:
101  *
102  *\li #ISC_R_SUCCESS
103  *\li #ISC_R_NOMEMORY
104  */
