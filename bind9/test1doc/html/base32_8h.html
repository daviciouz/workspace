Decode a packed (no white space permitted) region in base32, base32hex or base32hex non-padded.

Requires:

  • 'source' is a valid region.
  • 'target' is a valid buffer.

Returns:

  • #ISC_R_SUCCESS – the entire decoded representation of 'cstring' fit in 'target'.
  • #ISC_R_BADBASE32 – 'source' is not a valid base32 encoding.

Decode a null-terminated string in base32, base32hex, or base32hex non-padded.

Requires:

  • 'cstr' is non-null.
  • 'target' is a valid buffer.

Returns:

  • #ISC_R_SUCCESS – the entire decoded representation of 'cstring' fit in 'target'.
  • #ISC_R_BADBASE32 – 'cstr' is not a valid base32 encoding. 
    Other error returns are any possible error code from:
  • isc_lex_create(),
  • isc_lex_openbuffer(),
  • isc_base32_tobuffer().
isc_result_t isc_base32hexnp_tobuffer ( isc_lex_t *  lexer,
isc_buffer_t *  target,
int  length 
)

Convert text encoded in base32, base32hex, or base32hex non-padded from a lexer context into data.

Requires:

  • 'lex' is a valid lexer context
  • 'target' is a buffer containing binary data
  • 'length' is an integer

Ensures:

  • target will contain the data represented by the base32 encoded string parsed by the lexer. No more than length bytes will be read, if length is positive. The 'used' pointer in target will be advanced as necessary.

int  wordlength,
const char *  wordbreak,

Convert data into base32 encoded text.

Notes:

  • The base32 encoded text in 'target' will be divided into words of at most 'wordlength' characters, separated by the 'wordbreak' string. No parentheses will surround the text.

Requires:

  • 'source' is a region containing binary data
  • 'target' is a text buffer containing available space
  • 'wordbreak' points to a null-terminated string of zero or more whitespace characters

Ensures:

  • target will contain the base32 encoded version of the data in source. The 'used' pointer in target will be advanced as necessary.
