Detailed Description

template<class IS>
class atf::parser::tokenizer< IS >

A stream tokenizer.

This template implements an extremely simple, line-oriented stream tokenizer. It is only able to recognize one character-long delimiters, random-length keywords, skip whitespace and, anything that does not match these rules is supposed to be a word.

Parameter IS: The input stream's type.
